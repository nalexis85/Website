List of Terms Week 1:

  1. htmlHyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page's words and images.
  2. ulunordered list. The ul element represents an unordered list of items; that is, a list in which changing the order of the items would not change the meaning of list.
  3. imgThe Image Embed element. The img HTML element embeds an image into the document.
  4. titleThe title tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab. The title tag is required in HTML documents!
  5. urlURL stands for Uniform Resource Locator. A URL is nothing more than the address of a given unique resource on the Web.
  6. faviconA favicon is a graphic image (icon) associated with a particular Web page and/or Web site. Many recent user agents (such as graphical browsers and newsreaders) display them as a visual reminder of the Web site identity in the address bar or in tabs.
